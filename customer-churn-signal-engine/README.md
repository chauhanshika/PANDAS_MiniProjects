# Customer Churn Signal Engine

## Problem Statement
Customer churn is a major challenge for businesses. Losing customers without understanding early warning signals leads to revenue loss and poor retention strategies.

This project focuses on identifying early churn signals using user activity data.

## Objective
- Analyze user activity patterns
- Detect inactivity trends
- Generate churn risk indicators using Pandas
